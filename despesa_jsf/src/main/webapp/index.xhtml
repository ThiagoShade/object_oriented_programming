Esse é um projeto feito para a disciplina de Programação Orientada a Objetos usando padrão MVC com o JSF para a implementação de um sistema de nossa escolha, nesse caso um sistema de gerenciamento de despesas, sendo possível apresentar as despesas de várias formas, além de adicionar, excluir e editar despesas.
